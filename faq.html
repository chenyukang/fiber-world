Frequently Asked Questions

Find answers to common questions about Fiber Network

What is Fiber Network?

Fiber Network is a Layer 2 payment network built on CKB that enables instant, low-cost transactions through off-chain payment channels. It's designed to scale Satoshi's P2P electronic cash vision with enhanced privacy and interoperability features.

How does Fiber differ from Lightning Network?

While both use payment channels, Fiber Network supports multi-asset payments (CKB, RGB++ tokens), has O(1) watchtower storage complexity, and provides native cross-chain interoperability with Bitcoin Lightning Network through edge nodes.

What are RGB++ tokens?

RGB++ tokens are a new type of digital asset that can be issued and transferred on the CKB network. They provide enhanced privacy features and can seamlessly integrate with the Fiber Network for instant payments.

How do cross-chain atomic swaps work?

Edge nodes in the Fiber Network enable secure, trustless exchanges between different blockchain networks. Users can swap assets between Lightning Network and Fiber Network without requiring centralized exchanges.

What is the watchtower storage advantage?

Fiber Network's O(1) watchtower storage means that storage requirements remain constant regardless of the number of channels, making it more efficient than Lightning Network's O(n) storage that grows linearly with channel count.

When will mainnet be available?

Fiber Network mainnet is planned for Q1 2025. The current testnet is available for developers and early adopters to test the functionality and provide feedback.

How can I get involved?

You can contribute to the project through our GitHub repository, join our community discussions, test the network on testnet, or integrate Fiber Network into your applications using our developer tools.

What wallets support Fiber Network?

Wallet integrations are currently in development. We're working with major CKB wallet providers to ensure seamless Fiber Network support upon mainnet launch.
